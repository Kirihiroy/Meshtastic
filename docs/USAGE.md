# Использование приложения

## Подключение по BLE

1. Откройте вкладку "Подключение".
2. Нажмите "Поиск устройств".
3. Выберите устройство Meshtastic из списка.
4. Нажмите "Подключиться".

## Статус

Вкладка "Статус" показывает:

- состояние соединения
- последнюю сводку FromRadio
- версию прошивки
- батарею, SNR и время последнего приема
- последние входящие байты (hex)

## Настройки

Вкладка "Настройки":

- сохраняет черновик локально
- позволяет применить имя канала и PSK на устройство

Минимально требуется указать:

- Имя канала
- PSK
