## Экран "Подключение"

### 01-scan.png
**Описание:** Начало сканирования BLE устройств
**Содержимое:**
- Кнопка "Поиск устройств" активна
- Статус "Bluetooth включен. Готов к поиску устройств."
- Прогресс бар видим
- Список устройств пуст

**Когда снимать:** После нажатия "Поиск устройств"

---

### 02-device-list.png
**Описание:** Результаты сканирования
**Содержимое:**
- Список найденных устройств (1-3 устройства)
- Для каждого: имя и MAC адрес
- Кнопка "Подключиться" активна
- Прогресс бар скрыт

**Когда снимать:** Через 6 секунд после начала сканирования

---

### 03-connected.png
**Описание:** Успешное подключение
**Содержимое:**
- Статус "Подключено: Meshtastic_xxxx" зелёным
- Поле "Последние входящие байты" с hex данными
- Кнопка "Отправить тест" активна

**Когда снимать:** Сразу после успешного подключения

## Технические требования

### Формат
- **Тип:** PNG (для прозрачности и чёткости)
- **Разрешение:** 1080x2340 (стандарт для большинства телефонов)
- **DPI:** 420 (xxhdpi)

### Телефон для скриншотов
- **Модель:** Google Pixel или аналог с чистым Android
- **Версия Android:** 12+
- **Размер экрана:** 6.0-6.5 дюймов

### Как сделать скриншот

#### Через Android Studio
```
Run → Screenshot
Или: View → Tool Windows → Device File Explorer → screenshot
```

#### Через ADB
```powershell
adb shell screencap -p /sdcard/screenshot.png
adb pull /sdcard/screenshot.png ./docs/screenshots/
```

#### На устройстве
```
Power + Volume Down (одновременно)
Файл сохраняется в /Pictures/Screenshots/
```

## Чек-лист перед созданием скриншотов

- [ ] Приложение собрано в Release режиме (для корректных иконок)
- [ ] Устройство Meshtastic включено и рядом
- [ ] Bluetooth и геолокация включены
- [ ] Тестовые данные готовы (несколько узлов в сети)
- [ ] Батарея телефона > 50% (для чистого status bar)
- [ ] Время на телефоне 10:00 или 14:00 (красивое симметричное)
- [ ] Яркость экрана 100%
- [ ] Режим "Не беспокоить" включён (без уведомлений)
- [ ] Язык интерфейса: русский (или английский для международной документации)

### Для Google Play Store

Требования Google Play:
- **Минимум:** 2 скриншота
- **Максимум:** 8 скриншотов
- **Формат:** JPG или PNG (24-bit)
- **Размер:** 320-3840 px (любая сторона)
- **Рекомендуемый:** 1080x1920 (portrait)
