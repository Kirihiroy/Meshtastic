# Установка и требования

## Требования

- Windows 10/11
- Android Studio (актуальная версия)
- Android SDK (API 24+)
- JDK 11 (для Gradle)

## Переменные окружения

- `JAVA_HOME` должен указывать на путь к JDK 11
- `ANDROID_SDK_ROOT` должен указывать на каталог Android SDK

## Сборка

В PowerShell из корня проекта:

```powershell
.\gradlew.bat :app:assembleDebug
```

## Запуск на устройстве

1. Подключите устройство по USB.
2. Включите режим разработчика и USB отладку.
3. Запустите приложение из Android Studio.

## Примечания

- BLE в эмуляторе поддерживается ограниченно. Используйте реальное устройство.
