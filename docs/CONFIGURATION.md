# Настройки сети и безопасности

## Канал и PSK

Применение канала и PSK реализовано через отправку admin-пакета по BLE:

- порт: `ADMIN_APP`
- payload: protobuf `Channel` (index=0, role=PRIMARY, settings.name, settings.psk)

Текущий UI позволяет указать:

- имя канала
- PSK (строка, отправляется как bytes UTF-8)

## Черновик

Все поля сохраняются локально в `SharedPreferences` и могут применяться повторно.

## Ограничения MVP

- регион пока не применяется
- node name пока не применяется
- нет ответа об успехе от устройства
