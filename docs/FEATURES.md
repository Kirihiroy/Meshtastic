# Функции

## Реализовано

- BLE сканирование и подключение
- Мониторинг состояния устройства и сети
- Визуальная и текстовая обратная связь (статус, last RX, сводка)
- Список узлов сети
- Настройки канала и PSK с применением по BLE
- Локальный черновик настроек

## В работе

- Отображение RSSI/MTU в статусе
- Подтверждения и ошибки по admin/app ответам

## Планируется

- Экран чата
- Экран карты
- Расширенные настройки (region, device, power)
